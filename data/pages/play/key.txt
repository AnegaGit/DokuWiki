====== Control Keys ======
See also [[mouse|mouse controls]].
==== Movement ====
^Key^Effect^
|W, arrow up|forward|
|S, arrow down|backward|
|A, arrow left|counterclockwise rotation|
|D, arrow right|clockwise rotation|
|STRG left and right|creep|
|Shift left and right|run|
|Alt left|stay / stop|
==== UI ====
^Key^Effect^
|F1|Help|
|Y|Toggle camera fixation|
|I|Character information|
|E|Equipment|
|B|Backpack|
|Numpad /|Split stack control|
==== Selection ====
^Key^Effect^
|TAB|Select monster closest to you|
|ESC|Unselect current selection\\ Stops any action like attacking or trading.|
==== Miscellaneous ====
^Key^Effect^
|Shift on Drag&Drop|Split stacks|
|P|Pick all items in range (plunder)|
==== GM, Test and Debug ====
^Key^Effect^
|G|GM Control panel|
|F2|GM security key|
|K|Kill yourself (suicide)|
|F5|Test and Debug panel|
